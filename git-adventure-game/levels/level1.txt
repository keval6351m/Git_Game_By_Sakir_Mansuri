=== LEVEL 1: BEGIN YOUR JOURNEY ===

GOAL: Make your first commit!

TASKS:
1. Set your name: 
   git config user.name "Your Name"
   
2. Set your email:
   git config user.email "you@example.com"
   
3. Initialize Git:
   git init
   
4. Create story file:
   echo "Once upon a time..." > story.txt
   
5. Stage the file:
   git add story.txt
   
6. Commit it:
   git commit -m "Begin adventure"

COMPLETE WHEN:
echo "LEVEL1_DONE" > level1.check
